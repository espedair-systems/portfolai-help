Manage the visual and interactive aspects of your PortfolAI ecosystem.

## General System

The General System tab contains preferences for numerical formatting, notifications, and preview behaviors.

### Abbreviate Counters
Large numbers (e.g., file counts, view counts) are formatted to be more readable.
- **Enabled**: 1,500 becomes **1.5K**.
- **Disabled**: Shows the full value.

### System Notifications
Toggle desktop alerts for background event completions, such as when a large media scan finishes or an AI analysis is ready.

### Preview Behaviors
- **Hover Audio**: Plays the audio track of video files when hovering over their cards in the gallery.
- **Default Preview Volume**: Sets the initial volume for hover playback.
- **Playback Entry Mode**: Controls how the full-screen player starts (e.g., Always Pause or Always Start from Beginning).

---

## Filtering

Control the visibility of core system modules. Disabling a module removes it from the navigation menu.

### Storage Interface
The "Storage" section in the sidebar, providing access to partition management and disk health.

### Gallery Engine
The main gallery views and collection explorers.

### Global Search
The search bar and advanced filtering discovery tools.

### Control Panel Access
Access to the detailed settings and administrative configuration tabs.

:::tip[Admin Access]
If you accidentally disable the "Control Panel Access" and lose the settings link, you can navigate directly to `/settings` in your browser.
:::
